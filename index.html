Cheat-Sheet
Tag Name Description
<table> Table Information presented in a two-dimensional table comprised of rows and columns of cells containing data
<thead> Table Head Used to group header content in an HTML table.
<tbody> Table Body Used to group the body content in an HTML table
<tr> Table Row Defines a row of cells in a table
<td> Table Data Defines a cell of a table that contains data
<tfoot> Table Footer Defines a set of rows summarizing the columns of the table
<th> Table Header Defines a cell as header of a group of table cells
Tag Name Description
colspan Column Span Defines the number of columns a table cell should span
rowspan Row Span Specifies the number of rows a cell should span
Tag Name Description
Padding: Padding Padding creates extra space within an element
Margin: Margin Create space around elements, outside of any defined borders
Border: Border Defines the width, line style, and color of the border of a box
Positioning: Positioning Sets how an element is positioned in a document
